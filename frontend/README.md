# LMS Frontend

Next.js frontend for the Learning Management System.

## Quick Start

1. Install dependencies:
```bash
npm install
```

2. Set environment variable (optional):
```bash
# Create .env.local
NEXT_PUBLIC_API_URL=http://localhost:8000/api
```

3. Run development server:
```bash
npm run dev
```

## Features

- Authentication (Login/Register)
- Instructor Dashboard
- Course Management
- Chapter Editor with Plate.js
- Student Course Browser
- Chapter Viewer

## Tech Stack

- Next.js 14 (App Router)
- React 18
- TypeScript
- Tailwind CSS
- Plate.js (Rich Text Editor)
- Axios (HTTP Client)
